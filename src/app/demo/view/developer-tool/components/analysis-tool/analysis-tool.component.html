플레이어의 기능과 디자인을 컨트롤하는 파라미터는 플레이어 Parameters 문서를 참고하시기 바랍니다.

동영상의 삽입은 아래의 Play URL을 포함한 iframe 코드를 작성하여 웹페이지에
배포 패키지에 포함된 특정 동영상을 재생하는 플레이어를 삽입할 수 있습니다.
